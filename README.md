# DGUT 优学院助手

适配东莞理工学院优学院平台（ua.dgut.edu.cn）的 Tampermonkey 用户脚本。
支持自动静音播放、自动答题、自动翻页、自定义倍速播放，并提供悬浮控制面板。

## 功能一览

- ▶️ 自动播放视频（静音 + 自定义倍速 1x–15x）
- ✅ 自动答题（单选题、多选题、判断题、填空题，自动从接口获取答案）
- 📄 自动翻页 / 跳转下一节
- 🛡️ 防检测机制（playbackRate 劫持、鼠标模拟）
- 🎛️ 悬浮控制面板（深色主题、开关控制、实时日志、进度追踪）
- ⚙️ 配置自动保存（刷新页面不丢失设置）

## 保姆级安装教程

### 第一步：安装 Tampermonkey 浏览器扩展

> Tampermonkey（篡改猴）是一个浏览器扩展，用来运行用户脚本。
> 以下以 **Edge 浏览器** 为例，Chrome 浏览器操作基本一致。

1. 打开电脑上的 **Edge 浏览器**（或 Chrome 浏览器）
2. 访问 Edge 扩展商店：[点击进入 Microsoft Edge 加载项](https://microsoftedge.microsoft.com/addons/)
3. 在搜索框中输入 `Tampermonkey` 并回车
4. 找到 **Tampermonkey**（图标是一个黑色猴子，开发者是 Jan Biniok），点击「获取」→「添加扩展」
5. 安装完成后，浏览器右上角会出现一个 🐵 图标，说明安装成功

> 备用方法：也可以直接访问 Tampermonkey 官网 https://www.tampermonkey.net/ 点击下载安装。

### 第二步：安装本脚本

1. 打开本仓库，点击 `DGUT-Ulearning-Helper-v2.user.js` 文件
2. 点击文件右上角的「Raw」按钮
3. 浏览器会自动弹出 Tampermonkey 安装页面，点击「安装」即可

### 第三步：开始使用

1. 打开浏览器，访问东莞理工学院优学院课程页面：
   `https://ua.dgut.edu.cn/learnCourse/learnCourse.html`（需先登录）
2. 进入任意课程的学习页面
3. 页面右侧会出现一个深色悬浮面板，标题为 **DGUT 优学院助手 v2.1**
4. 脚本默认已自动运行，无需额外操作

### 第四步：控制面板使用说明

**面板布局：**

- **顶部标题栏**：拖动可移动面板位置；点击「▼」可折叠/展开
- **状态栏**：显示「运行中」或「已暂停」；点击「暂停/继续」按钮控制脚本启停
- **播放速率**：点击「+」「−」调整倍速（范围 1x–15x）
- **视频设置**：开关控制自动播放、自动静音、自动调整速率
- **自动作答**：开关控制自动答题、显示答案及各题型开关
- **保存设置**：修改开关后必须点击此按钮才能生效
- **运行日志**：显示脚本实时运行状态，点击「清空」可清除日志

> **提示**：所有设置会自动保存到浏览器本地，下次打开页面自动恢复。

## 技术栈

- Tampermonkey API（GM_xmlhttpRequest）
- 原生 JavaScript（ES6+，Class 结构）
- jQuery（依赖优学院页面已加载）
- CSS 深色主题 UI
